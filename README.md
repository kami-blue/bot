## Usage

```
git clone git@github.com:kami-blue/bot.git kamiBlueBot
cd kamiBlueBot
npm i
node index.js
```

Requirements: 
```
node >=12
git(latest)
go(latest)
GNU make(latest)

//on Linux
Run the install.sh file with root.

//on Windows
run the install.bat file with root and install https://github.com/buildkite/terminal-to-html manually.

//on MacOS
Run the install.tool file 2 times, with and without root.
```

`auth.json` example:
```json
{
  "token": "tokenhere",
  "githubtoken": "tokenhere"
}
```
